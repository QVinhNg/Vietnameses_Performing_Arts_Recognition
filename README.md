# Vietnamese Performing Arts Recognition

## Project Description
This project aims to recognize and describe various forms of Vietnamese performing arts from images. It utilizes MobileNet for feature extraction and integrates a large language model to generate detailed descriptions of the recognized art forms.

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- FAISS
- Gradio
- Google Generative AI

## Installation

To set up the project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd Vietnameses_Performing_Arts_Recognition
   ```

2. **Install the required packages:**
   It is recommended to use a virtual environment. You can create one using `venv` or `conda`. Then, install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. **Run the application:**
   To start the application, use the following command:
   ```bash
   python app.py
   ```

2. **Upload an image:**
   Once the application is running, you can upload an image of a Vietnamese performing art form through the web interface.

3. **Get the description:**
   The application will process the image and provide a description of the recognized art form.
